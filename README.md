SupportIQ — Smart Contact Center Knowledge Management System

SupportIQ is a full-stack web application designed to improve customer support efficiency through AI-assisted troubleshooting, a structured knowledge base, and an administrative dashboard for managing support workflows.

The platform allows users to resolve issues through an AI chatbot or guided decision trees, while administrators can manage categories, questions, answers, and knowledge base entries through a secure admin panel.

✨ Key Features

--Customer Support Interface--

* AI chatbot for natural language support queries
* Guided troubleshooting through decision-tree flows
* Searchable knowledge base for quick answers
* Contact form for customer messages

--Admin Dashboard--

* Secure admin login using JWT authentication
* Create and manage support categories
* Build decision-tree troubleshooting flows
* Manage knowledge base entries for chatbot responses
* View and manage customer messages
* Update message status (New / Read / Resolved)

--Backend System--

* RESTful API architecture
* MongoDB database integration
* Token-based admin authentication
* Role-based admin route protection

⚙️ How It Works

1. Users interact with the AI chatbot or browse troubleshooting categories.
2. Decision trees guide users through step-by-step solutions.
3. Knowledge base entries allow the chatbot to answer common questions.
4. Admin dashboard allows administrators to manage all support content.
